# Arduino-servo-description
This is the description for controlling your 4 servos with a Arduino uno and a PCA9685 servo driver board.
First before you run the code you need to install the librarys. Step 1, open your arduino IDE. Step 2, go to "sketch" up top when you get to the dropdown go to include>Manage librarys.
Step 3, At the search bar up top search "adafruit servo driver". After you type that in there should be a bar that pops up and says, Adafruit PWM Servo Driver Library. If you hover over it there will be a button that says "instal" press that button. after it installs you can press close. Now you are ready to code your arduino!
